This containes programs written in Hack Assembly, a language that is provided for Hack CPU by the course.
   You can run those using provided CPU emulator (it automatically compiles it into machine code).
   - Mult: Simple multiplication program, it multiples value from RAM[0] by RAM[1] and saves the outcome to RAM[2]
   - Fill: Pressing and holding any key should make the screen black, freeing the key should make it white again.
